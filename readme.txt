== Description ==

Simple plugin which wraps a Twitter widget as a Wordpress widget.

In order to use it, you need the Twitter widget's ID - something like
332108549298064896. You can see it in the embeddable code Twitter gives you
after you create the widget. You also need the Twitter username for the account
the widget belongs to.

== Installation ==

1. Unzip the archive to a folder in your '/wp-content/plugins' folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Twitter user name and the widget's ID
